## 기능 명세

### 6.11 음식점 등록
- 음식점명, 카테고리, 대표 사진 업로드
- 최소 주문금액, 배달비, 예상 배달 시간
- 영업시간 설정
- 카카오 우편번호 → 위도/경도 변환 → 지도 핀 확인
- **메뉴 빠른 시작 (등록 마지막 단계):**
  1. **카테고리 템플릿 적용** — 대표 메뉴 + 옵션 자동 생성 (가격만 입력)
  2. **다른 가게 메뉴 복사** — 프랜차이즈/인기 가게 메뉴 구조 복사
  3. **처음부터 직접 등록**
- 등록 완료 → role USER → OWNER
- 관리자 승인 없이 즉시 노출

### 6.12 주문 관리
- 신규 주문 실시간 수신 (Centrifugo `owner_orders#<ownerId>` 채널)
- **주문 상태 변경:**
  - 접수 (PENDING → COOKING)
  - 배달 요청 (COOKING → WAITING_RIDER) — 근처 기사 브로드캐스트
  - 거절 (PENDING → CANCELLED, 사유 필수)
- 주문 상세: 메뉴, 수량, 옵션, 고객 주소
- 배달 매칭 상태: 기사 찾는 중 / 배정됨 (기사 정보)
- 취소 사유: 재료 소진 / 영업시간 종료 / 주문 폭주 / 기타

### 6.13 메뉴 관리
- 메뉴 등록: 이름, 카테고리, 가격, 설명, 이미지
- **메뉴 옵션 관리:** 옵션 그룹 + 옵션 항목 CRUD
- 메뉴 수정/삭제, 품절 ON/OFF
- **메뉴 가져오기:**
  - 카테고리 기본 템플릿 (가격 0원 → 입력)
  - 프랜차이즈 복사 (같은 이름 가게 검색, 구조만 복사)
  - 인기 가게 참고 (같은 카테고리)
- **PC 테이블 레이아웃:** 썸네일 | 메뉴명 | 카테고리 | 가격 | 품절 토글 | 뱃지 | 수정/삭제
- 카테고리별 탭, 드래그 순서 변경, 우측 슬라이드 패널 등록

**메뉴 템플릿 데이터 (시스템 내장, 시드):**

| 메뉴명 | 옵션 그룹 | 옵션 항목 |
|--------|----------|----------|
| 후라이드치킨 | 사이즈 (필수, 택1) | 한마리 (+0) / 반마리 (-5,000) |
| | 부위 (선택, 택1) | 순살 (+2,000) / 뼈 (+0) |
| 양념치킨 | 사이즈 (필수, 택1) | 한마리 (+0) / 반마리 (-5,000) |
| 간장치킨 | 사이즈 (필수, 택1) | 한마리 (+0) / 반마리 (-5,000) |
| 반반치킨 | 조합 (필수, 택2) | 후라이드 / 양념 / 간장 / 매운양념 |
| 치킨텐더 | - | - |
| 치즈볼 | - | - |
| 콜라 | 사이즈 (선택, 택1) | 500ml (+0) / 1.5L (+1,000) |

### 6.13-1 사장 대시보드 (PC 전용)

**전체 레이아웃:**
```
┌──────────────────────────────────────────────────────────────────┐
│  🟢 영업중 ▼  │  B-Delivery 사장님  │  가게명  │  알림 🔔  │  설정 ⚙  │
├────────┬─────────────────────────────────────────────────────────┤
│ 사이드바 │  ┌─────────┬──────────┬──────────┬──────────┐         │
│ 📊 대시보드│  │ 대기(3)  │ 조리중(2) │ 배달대기(1)│ 배달중(1) │         │
│ 📋 주문관리│  ├─────────┼──────────┼──────────┼──────────┤         │
│ 🍽 메뉴관리│  │ 주문카드  │ 주문카드   │ 주문카드   │ 주문카드   │         │
│ ⭐ 리뷰관리│  └─────────┴──────────┴──────────┴──────────┘         │
│ 📈 매출통계│  ┌──────────────────┬───────────────────────────┐     │
│ 💬 채팅   │  │ 오늘 매출 요약     │ 시간대별 주문 추이 (차트)     │     │
│ ⚙ 가게설정│  └──────────────────┴───────────────────────────┘     │
│        │  ┌──────────────────┬───────────────────────────┐       │
│        │  │ 인기 메뉴 TOP 5   │ 최근 리뷰 (미답변 우선)      │       │
│        │  └──────────────────┴───────────────────────────┘       │
└────────┴─────────────────────────────────────────────────────────┘
```

**1) 칸반 보드 (상단):** 대기 | 조리중 | 배달대기 | 배달중 4컬럼
- 건수 뱃지, 주문 카드 (번호, 메뉴 요약, 금액, 경과 시간)
- 3분 초과 시 빨간색 강조
- 액션 버튼: 접수/거절 | 배달요청
- 신규 주문: 알림음 + 애니메이션 + 브라우저 Notification
- 카드 클릭 → 우측 슬라이드 패널 (주문 상세)

**2) 오늘 매출 요약:** 실시간 매출, 주문 건수, 평균 금액, 평균 배달 시간 + 전일 대비 증감

**3) 시간대별 주문 추이:** 막대 그래프 00~23시, 현재 하이라이트

**4) 인기 메뉴 TOP 5:** 순위 | 메뉴명 | 주문 수 | 매출 기여도

**5) 최근 리뷰:** 24시간 신규, 미답변 강조, 별점 추이, 바로 답글

**6) 영업 상태 토글:** 영업 중 / 종료 / 일시 중지 (주문 폭주), 확인 모달

**7) 매출 통계 페이지:** 기간 선택, 일별 그래프, 매출 테이블, 메뉴별 순위, CSV 다운로드

### 6.8.3 사장 채팅 (OWNER → ADMIN)
- 사이드바 채팅 메뉴에서 진입
- chatType: `OWNER_SUPPORT`
- 주문 관련: 주문 선택 → 고객 확인 요청 등
- 운영 관련: 가게 정보, 메뉴, 정산 문의
- 채팅 공통 UI (6.8.6) 사용

### 6.9.3 사장 답글
- 자기 음식점 리뷰에 답글 1개 작성/수정/삭제

## 주문 상태 전이 (사장 관점)

| 현재 상태 | 사장 액션 | 다음 상태 |
|-----------|----------|----------|
| PENDING | 접수 | COOKING |
| PENDING | 거절 (사유 필수) | CANCELLED |
| COOKING | 배달 요청 | WAITING_RIDER |
| COOKING | 취소 (사유 필수) | CANCELLED |

## Centrifugo 채널 (사장이 구독)

| 채널 | 용도 | 구독 방식 |
|------|------|----------|
| `owner_orders#<ownerId>` | 신규 주문 알림, 배달 매칭 결과 | Connect Proxy (서버 사이드) |
| `user#<userId>` | 채팅 알림 | Connect Proxy (서버 사이드) |
| `chat:<chatId>` | 채팅방 메시지 | Subscribe |
