# Task #25: 마이페이지 코어 UI 및 네비게이션

## 개요
마이페이지 메인 화면의 코어 UI를 완성하고, 요구사항에 맞는 네비게이션 링크를 구성한다.

## 현재 상태 분석
- `src/app/(customer)/mypage/page.tsx` 에 기본 마이페이지가 이미 구현되어 있음
- 프로필 섹션 (아바타, 닉네임, 프로필 수정 링크) 구현됨
- 기본 주소 표시 구현됨
- 쿠폰/포인트/선물함 섹션 구현됨
- 메뉴: 리뷰관리, 주소관리, 고객센터, 음식점 등록하기 존재

## 누락 항목 (요구사항 대비)
1. **주문 내역** 네비게이션 링크 누락
2. **관심 음식점** 네비게이션 링크 누락
3. 프로필 수정 버튼이 아이콘만 있어 접근성이 부족 → 텍스트 버튼으로 변경

## 구현 계획

### 변경 파일
- `src/app/(customer)/mypage/page.tsx` — 메뉴 항목 추가

### 추가할 메뉴 항목
| 메뉴 | 아이콘 | 링크 |
|------|--------|------|
| 주문내역 | ClipboardList | /orders |
| 관심 음식점 | Heart | /favorites |
| 리뷰관리 | Star | /mypage/reviews (임시) |
| 주소관리 | MapPin | /mypage/addresses |
| 고객센터 | MessageSquare | /mypage (임시) |
| 음식점 등록하기 | Store | /mypage/register-restaurant (USER만) |

### 디자인 참조 (ui/4.마이페이지.png)
- 프로필 영역: 상단 중앙에 아바타 + 닉네임
- 네비게이션: 리뷰관리, 주소관리 등이 리스트 형태로 표시
- 하단 네비게이션 바 존재 (별도 레이아웃 컴포넌트)
