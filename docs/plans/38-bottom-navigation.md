# Task #38: 글로벌 하단 네비게이션 바

## 개요
고객 페이지 전체에 표시되는 4탭 하단 네비게이션 바 구현/개선

## 현재 상태
- `src/app/(customer)/_components/bottom-nav.tsx`에 5탭 구현이 이미 존재
- 고객 레이아웃(`src/app/(customer)/layout.tsx`)에 이미 통합됨
- 카테고리 탭 제거 필요 (UI 스크린샷 기준 4탭)

## 변경 사항

### 탭 구성 (4탭)
| 탭 | 경로 | 아이콘 | 라벨 |
|----|------|--------|------|
| 홈 | `/` | `Home` | 홈 |
| 찜 | `/favorites` | `Heart` | 찜 |
| 주문내역 | `/orders` | `ClipboardList` | 주문내역 |
| 마이페이지 | `/mypage` | `User` | 마이페이지 |

### 기술 상세
- `lucide-react` 아이콘 사용 (이미 설치됨)
- `usePathname` 기반 활성 탭 하이라이트
- 채팅방/음식점 상세에서 숨김 처리 유지
- UI 스크린샷 참조하여 스타일 개선 (민트색 활성 상태)

### 스타일
- 활성 탭: 민트/틸(#00C4B4) 색상, 굵은 폰트
- 비활성 탭: 회색(muted)
- 하단 고정, 흰색 배경, 상단 보더
